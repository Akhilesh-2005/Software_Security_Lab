#  Software Security Lab

This repository contains **10 practical experiments** performed as part of the *Software Security Lab*. The experiments focus on identifying vulnerabilities, applying secure coding practices, and analyzing real-world security threats in web applications.

---

##  Repository Structure

---

##  List of Experiments

### 🔹 Experiment 1: Vulnerability Analysis & Secure Coding
- Analyze code to identify vulnerabilities such as buffer overflow and format string issues  
- Apply secure coding practices  
- Tools: C, Python, Static Code Analysis  

---

### 🔹 Experiment 2: SQL Injection Attack Simulation
- Simulate SQL Injection attacks on login system  
- Prevent attacks using prepared statements  
- Tools: PHP, MySQL, XAMPP  

---

### 🔹 Experiment 3: Secure Password Hashing
- Implement password hashing with salting  
- Use cryptographic techniques for secure storage  
- Tools: Python (hashlib)  

---

### 🔹 Experiment 4: Secure Code Review
- Review login system code  
- Identify vulnerabilities and suggest fixes  
- Focus on input validation and authentication security  

---

### 🔹 Experiment 5: Network Traffic Analysis
- Capture login data packets  
- Analyze HTTP communication  
- Demonstrate insecurity of plaintext transmission  
- Tools: Wireshark  

---

### 🔹 Experiment 6: Threat Modeling (STRIDE)
- Identify threats using STRIDE model  
- Categorize risks like spoofing, tampering, etc.  
- Suggest mitigation strategies  

---

### 🔹 Experiment 7: Secure Authentication System
- Develop login system with password hashing  
- Verify credentials securely  
- Tools: PHP, MySQL  

---

### 🔹 Experiment 8: Vulnerability Scanning (OWASP ZAP)
- Perform automated vulnerability scanning  
- Analyze alerts and generate reports  
- Tools: OWASP ZAP  

---

### 🔹 Experiment 9: Browser Security Analysis
- Inspect cookies, session storage, and local storage  
- Identify privacy and security issues  
- Tools: Chrome Developer Tools  

---

### 🔹 Experiment 10: Brute Force Attack Detection
- Analyze login attempts from logs  
- Detect repeated failed login attempts  
- Implement account lock mechanism  

---

## Technologies Used

- PHP & MySQL  
- Python  
- XAMPP  
- Wireshark  
- OWASP ZAP  
- Chrome DevTools

##  Learning Outcomes

- Understand common web vulnerabilities  
- Apply secure coding techniques

- Analyze network and browser-level security issues
